# kotlinPractice

**프로젝트 목적 및 특징**

- **코틀린**을 사용한 첫 프로젝트
- 호주에서 각종 **레스토랑에서 일한 경험으로 주방에서 필요한 서비스**를 만들자라는 생각에서 시작하게 됌
- 재고관리, 레스토랑 관리는 있지만 **주방**을 관리하는 앱은 없어서 필요성을 느낌
